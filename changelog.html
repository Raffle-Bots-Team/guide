Changelog

All notable changes for this software will be documented here.

[Unreleased]​

[0.7.0]​

Added​

  • Batch add raffles
  • Allow user to select a range of accounts to register raffles instead of using all accounts

Changed​

  • Remove wait time for already joined raffles
  • Skip Already closed raffles
  • Show raffle added time in edit raffles page
  • Update default and recommended time interval values

[0.6.2]​

Fixed​

  • Fix Premint raffle url parsing bug

[0.6.1]​

Fixed​

  • Fix Premint requirement parsing bug

[0.6.0]​

Added​

  • Link filter utility module

[0.5.0]​

Added​

  • Skip already followed twitter accounts while registering raffles
  • Allow user to set per-account email

[0.4.0]​

Added​

  • Allow users to set the time interval between raffles

[0.3.0]​

Added​

  • Allow users to set the time interval between following twitter accounts

Fixed​

  • Twitter oauth error for non-English twitter accounts
  • Processes are not killed in Windows

[0.2.0]​

Added​

  • Allow users to register raffles with pipeline mode

[0.1.1]​

Fixed​

  • Error when proxy url contains dash symbol

[0.1.0]​

Added​

  • ETH wallet generator
  • Proxy format converter
  • Premint raffle bot
    • Manage social connections (connect/disconnect twitter and discord)
    • Edit raffles
    • Register raffles
    • Check entry
  • Superful raffle bot
    • Manage social connections (connect/disconnect twitter and discord)
    • Edit raffles
    • Register raffles
    • Check entry
