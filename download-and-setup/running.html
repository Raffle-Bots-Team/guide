Running

Our software does not have any malicious behaviors. However, if you are worried about security, you can always run our software in a virtual machine. Expand this collapsed section to see some articles about how to setup virtual machines. You can also find good tutorial videos on Youtube. After you successfully setup a virtual machine, you can follow the corresponding running guide for each operating system.
Note: If you don't want to use virtual machine, you don't need to expand this section.

Amazon EC2​

https://docs.aws.amazon.com/AWSEC2/latest/WindowsGuide/EC2_GetStarted.html

Amazon EC2 is a cloud computing service offered by Amazon. New accounts get 1 year free usage and can use EC2 750 hours per month (about 1 instance running 24/7). You can create Windows virtual machine on it. However, the machine that it provides does not have too much memory and CPUs, so it may be laggy.

  • Pros:
    • New accounts get 1 year free usage of one EC2 virtual machine instance.
    • You can create Windows virtual machines on it.
  • Cons:
    • You need to pay money when the free trial expires.
    • The virtual machine that the free tier provides does not have too much memory and CPUs so it may be laggy.

Google Cloud Platform​

https://cloud.google.com/architecture/chrome-desktop-remote-on-compute-engine

Google Cloud Platform is a cloud computing service offered by Google. New accounts get $300 free credits and 90 days free trial period. You can create virtual machines on it. However, under free tier you cannot create Windows machine. So here we provide a tutorial about how to setup a Linux virtual machine.

  • Pros:
    • New accounts get $300 free credits so you can run a virtual machine with better hardware, resulting a smoother experience.
  • Cons:
    • You need to pay money if you run out your free credits or when the free trial expires.
    • Not all people are familiar with Linux system.

VirtualBox​

https://www.groovypost.com/howto/windows-10-install-virtualbox/

VirtualBox is a software that can run virtual machines in our local machines. Remember that virutal machines running in local machine requires much memory. If your computer does not have much memory, then your system will be very laggy when running the virtual machine.

  • Pros:
    • Free
  • Cons:
    • If your computer does not have much memory, then whole system will be very laggy.

Windows​

  1. Execute your downloaded Mass Raffle installer
  2. Windows may recognize this app as an unknown app, click Run anyway.
  3. Click Install and wait for the installation completes
  4. Now Mass Raffle is successfully installed on your computer

MacOS​

  1. Navigate to your downloaded Mass Raffle .dmg file
  2. Double-click the .dmg file
  3. Drag-and-drop the Mass-Raffle.app icon to your Applications folder
  4. Press cmd + spacebar to open Spotlight
  5. Search Terminal.app and press enter
  6. Copy and paste the following command to the terminal:
    xattr -d com.apple.quarantine /Applications/Mass-Raffle.app
  7. Press enter
  8. Close the terminal
  9. Double-click the installed Mass-Raffle.app
  10. Enjoy!

Linux​

  1. Please go to your downloaded Mass Raffle AppImage and open a terminal there
  2. Run chmod +x Mass-Raffle-*.AppImage to change its permission to allow it to be executed
  3. Double click the file to execute it.
